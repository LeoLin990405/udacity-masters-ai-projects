## Overview

This repository is the canonical index for my Udacity x Woolf University Master's in AI project work.

It serves two purposes:

1. Provide one stable portfolio entry point for the full program.
2. Route readers to the strongest standalone project repositories first.

The portfolio currently spans 12 projects across 4 nanodegrees:

- AI Programming with Python
- Machine Learning with PyTorch
- Data Analyst
- Statistics for Data Analysis
